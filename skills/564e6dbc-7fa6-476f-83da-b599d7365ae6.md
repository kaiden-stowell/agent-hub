Send an iMessage using macOS Messages app.

## Instructions

When the user asks to text or message someone, run this AppleScript:

```bash
osascript <<'EOF'
tell application "Messages"
    set targetService to 1st account whose service type = iMessage
    set targetBuddy to participant "PHONE_NUMBER" of targetService
    send "MESSAGE_BODY" to targetBuddy
end tell
EOF
```

Replace `PHONE_NUMBER` with the recipient's number in E.164 format and `MESSAGE_BODY` with the message content.

If arguments are provided: `$ARGUMENTS[0]` is the phone number, and everything after is the message body.
